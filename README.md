# 📚 Student Management System

## 🚀 Overview

This is a **Python-based Student Management System** designed to manage student records efficiently.
It stores student data such as personal details, class information, and parent details using **JSON file handling**.

---

## 🛠️ Features

### ✅ Implemented Features

* Add new student
* Store student data in JSON format
* Manage parent information

---

### ⚠️ Missing Features (Work in Progress)

* ❌ Update student details
* ❌ Delete student records
* ❌ Search functionality

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/student-management-system.git
cd student-management-system
```

### 2. Create virtual environment

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

```bash
python main.py
```
